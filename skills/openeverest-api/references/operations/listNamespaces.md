# GET /namespaces

**Resource:** [General info](../resources/General-info.md)
**Managed namespaces**
**Operation ID:** `listNamespaces`

This API returns a list of namespaces managed by Everest.


## Responses

| Status | Description |
|--------|-------------|
| 200 | Successful operation |

**Success Response Schema:**

[NamespaceList](../schemas/Namespace/NamespaceList.md)
